                                                # 1. 算法评估

                                                一个操作如果和样本的数据量没有关系，每次都是固定时间完成的操作，叫做常数操作。

                                                一般来说，评价一个算法的好坏，先看时间复杂度，然后再分析常数操作所需要的时间。

                                                # 算法时间评估模型

                                                  1. 每个元素最多会被访问多少次
                                                  1. 循环中的时间复杂度找到循环中量的变化，与做事情的次数
                                                  1. MASTER公式

                                                # 递归模型

                                                  1. 放苹果模型,不断把一件事情变成简单的事情，最后在简单的事情上进行累加。
                                                  1. 汉诺塔模型，不断做一件事情。
                                                  1. 二叉树的各种递归。三种

                                                  • （找公共结点）利用后序遍历向上返回信息的特性，从上方获取下面的数据。

                                                  • （序列化） 利用遍历，在打印的位置新建string，并在最后返回stirng。储存性的递归。

                                                  • （折纸） 利用单纯的递归，模仿了二叉树的中序遍历，利用，i，N，bool三个参数完成。

                                                # 2. 位运算

                                                # 2.1   异或的性质

                                                0∧N=N 0\wedge N=N 0∧N=N

                                                N∧N=0 N\wedge N= 0 N∧N=0

                                                由上述两个式子，得知异或满足交换律和结合律，用没有脑子的话说就是：

                                                  1. 可以换位置,即 N∧0=0∧N=1N\wedge 0= 0\wedge N = 1N∧0=0∧N=1 。
                                                  1. 括号可以乱括起来,即 N∧(0∧N)=(N∧0)∧N=1N\wedge \left ( 0\wedge N \right ) = \left ( N\wedge 0 \right ) \wedge N = 1N∧(0∧N)=(N∧0)∧N=1 。

                                                有一种异或的说法就是不进位相加，半加器和全加器就有异或运算的参与。

                                                代码：

                                                int arr[6] = {1,1,2,2,3};  //样本  
int len = sizeof(arr)/sizeof(int); //计算数组的长度
int err=0; //用来存储异或的结果

for(int i=0;i<len;i++)
	{
		err^= arr[i];   //计算所有数一起异或后的异或值
	}

                                                err则为问题的答案。

                                                # 2.2   用法

                                                  1. 问一种奇次数的数，其余都是偶次数的数，求奇次数的数。

                                                例子：

                                                f(1,1,2,2,3)⇒3 f\left ( 1,1,2,2,3 \right ) \Rightarrow 3 f(1,1,2,2,3)⇒3

                                                经过异或的性质就可以知道，把所有的数都异或，就可以得出3。

                                                (1∧1)∧(2∧2)∧3=0∧0∧3=0∧3=3 \left ( 1\wedge1 \right )\wedge\left ( 2\wedge 2 \right ) \wedge 3 = 0 \wedge 0 \wedge 3 = 0 \wedge3 = 3 (1∧1)∧(2∧2)∧3=0∧0∧3=0∧3=3

                                                  1. 问两种奇次数的数，其余都是偶次数的数，求奇次数的数。

                                                f(1,1,2,2,3,4)⇒3,4 f\left ( 1,1,2,2,3,4 \right ) \Rightarrow 3,4 f(1,1,2,2,3,4)⇒3,4

                                                根据1，我们有从一堆数中找到一种奇次数的方法。

                                                思维模式：

                                                分类的具体方法：

                                                所有的数异或后得出：A∧BA \wedge BA∧B

                                                A和B是两个不同的数，异或之后，为1的位，说明两个数在这一位不同，比如3∧43 \wedge 43∧4如下所示：

                                                00000011∧000000100=00000111(公式1) 00000011 \wedge 000000100 = 00000111 (公式1) 00000011∧000000100=00000111(公式1)

                                                根据上述等号右边的位，可以随便把所有的数分为两组，一组是该位有1，另一组则是该位无1。

                                                如果样本为1,1,2,2,3,4则可进行如下分类:

                                                按第一位分

                                                为1： 1，1，3 ⇒\Rightarrow⇒ 异或得： 3

                                                为0： 2，2，4 ⇒\Rightarrow⇒ 异或得： 4

                                                按第二位分

                                                为1： 2，2，3 ⇒\Rightarrow⇒ 异或得： 3

                                                为0： 1，1，4 ⇒\Rightarrow⇒ 异或得： 4

                                                按第三位分

                                                为1：4 ⇒\Rightarrow⇒ 异或得： 4

                                                为0：1，1，2，2，3 ⇒\Rightarrow⇒ 异或得： 3

                                                # 根据公式1和分组为0所得的数进行一次异或，可以得到另外一个分组的奇次项数字。

                                                即：

                                                公式1∧分组后为0=3∧4∧3or4=4or3 公式1 \wedge 分组后为0 = 3\wedge 4 \wedge 3or4 = 4or3 公式1∧分组后为0=3∧4∧3or4=4or3

                                                这里必须要为分组0，不是的话程序会出错，这是关于异或运算的一些细节。想先了解这个

                                                代码：

**代码**：

```c

a = a^b; //结果为 a^b

                                                致命缺陷：

                                                如果两个地址相同的数进行异或交换，会出现乱码。

                                                怎么会有人用地址相同的两个数进行交换呢？傻吧。

                                                会的，因为所有的算法都趋向与给你一个框架，然后代码自己跑

                                                他在一个框架中乱跑，会不会出现地址相同的两个数进行交换，就是你的框架问题了。

                                                # 2.4   补充，位运算

                                                # 上文有说到一个问题是==号左边最好是0 寻找上文

                                                	int a = 0b00000100;
	int b = 0b00001000; 
	printf("\r\n%d",a^b);

                                                上述代码的输出结果是12。

                                                也就是说，如果分组中有8，分组的位是第三位，

                                                那么这两个数进行异或运算是12，并不是1，

                                                它属于1的分组，但是异或运算的结果并不是1。

                                                而处于0的分组异或运算绝对是0。
