                                              # 1.欧姆定律

                                              这是最简单的一个东西，但是我觉得真正理解他确实有点困难。必须时刻思考，所有东西的阻抗是否足够影响到你需要的东西，因为所有东西都有电阻，电容，电感三个东西，并且这些东西都会分压，也会限制电流的大小。

                                              如果把上图的接地电压变成电容，就会变成下图：

                                              对这两个图有两个思考：

                                              • 1.两条曲线刚好是反过来的。在电压最高的时候，电流最小，电流最高的时候电压最小。 什么时候两者的乘积最大呢，即使两个相等的时候，也就是在1个τ之前的时间会达到这个峰值。
                                              • 2.时间是从0开始的，所有的器件都是没有充电的，也就是零状态响应。阶跃是瞬间产生的，电流也是瞬间跳变的。也就是说电容阻碍了电压的变化，但是电流没有受到影响。电流的变化是由于后来电容的特性，吸收电荷而减小的。

                                              电容是阻碍电压的变化，不阻碍电流的变化，但是电流不是没有变化的，电容会在电流对响应的变化完成之后，再对电流进行变化。

                                              # 4.电感阻碍电流的变化

                                              电感和电容是精确互补元件。所有东西都非常类似。

                                              电感是阻碍电流的变化，不阻碍电压的变化，但是电压不是没有变化的，电感会在电压对响应的变化完成之后，再对电压进行变化。

                                              # 6.戴维南定理

                                              戴维南定理指于从输出的地方看整体的电路。

                                              法则：电压源要短路，电流源要开路口。

                                              上面的电路可以化简为下面的电路。这在求RC中R的值时有一定的用处。戴维南定理是我们老师说学电路分析不能忘记的电路之一。

                                              # 1.恒定电流源和恒定电压源

                                              • 恒定电压源：电源输出到负载两端的电压试图维持不变。所有的供电设备都有功率，只有自己的功率达到的情况下，才能使电压的恒定。

                                              • 恒定电流源：电源输出到负载两端的电流试图维持不变。所有的供电设备都有功率，只有自己的功率达到的情况下，才能使电流的恒定。

                                              一般来说，由于电流源的设备相对来说比电压源的设备更容易坏，因为电流始终是罪魁祸首，所以，一般来说电压源比较常见。他们总想维持他们提供的电压和电流，这种维持根据自身能力大小来判定的。

                                              # 2.交流

                                              在交流电路中，电流重复地每隔一段时间就改变一次方向，并且电流大多以正弦的方式交替地改变电流方向。

                                              会变成这样的原因在于交流功率在磁与线圈旋转的情况下很容易产生，并且产生的是正弦的电流。文中在这里讲了一个爱因斯坦和特斯拉的小故事。

                                              # 3.电容与电感

                                              电容与电感可以看成是对频率响应的电阻。

                                              电容对频率的敏感是成反比的。 电感对频率的敏感是成正比的。

                                              根据几句简单的话，就可以分辨高通，低通滤波器这些东西了。在学校简直无法想象。

                                              # 3.1   低通滤波器和高通滤波器

                                              为什么这是一个低通滤波器呢，因为在频率低的时候电容的阻抗大，所以输出都是低频的。 就这么简单，下面我都不想讲高通滤波器了。你想怎么和电阻，电容，电感搭配都行，在理论的世界，通什么频率的电路都可以用这个方法来配对。

                                              # 3.2   带通和带阻滤波器

                                              • 带通滤波器：先通过低通滤波器，在通过高通滤波器。就会形成只有一个带状的频率比较容易穿过的电路。
                                              • 带阻滤波器：高通滤波器和低通滤波器并联，最后相耦合，就会形成一个带状频率难以穿过的电路。

                                              # 3.3   有源滤波器

                                              无源元件基本都会受到负载效应的干扰。如下图所示：

                                              当3.1的低通滤波器带上负载，进行戴维南等效之后，电容的充电系数就发生了改变。

                                              并且这种改变不能够被预判，因为负载是很可能不断变化的。因此有源滤波器登场：

                                              最初的想法很简单，就是在输出的线路上加上一个尽可能大的电阻，这样在和内部的电阻串联的时候就不会受到负载的干扰。

                                              尽可能大的电阻说的就是运放。我算是明白为什么模电讲完三极管就讲运放了。

                                              当然当运放提供的电阻的量级和负载接近的时候，负载也是不能够被忽略的。

                                              # 3.4   磁场

                                              磁场和电场的关系，基本就是电感和电容的关系。

                                              在电感关断的时候，如果电流没有地方流动，那么电感上的电压就会瞬间增高产生一个振荡，然后才落回正常电压。

                                              # 3.5   电场

                                              电流具有磁性，电压具有电性。

                                              电容即使一个把电荷聚集起来变成一定程度的电压，然后产生电场的元件。

                                              电容能够存储的电荷数量示极板面积的函数，而电容能够建立的数值则依赖于电介质的强度。

                                              其余特性和电感类似，也需要一定的泄放电路。
