                                    # 1. 钱不是资产

                                    对于经济，我刚起步，实在是说不出什么东西。

                                    钱本来就是定义的一个物品，钱的价值和国家的信用有关，和通货膨胀有关

                                    至今我也不明白什么是通货膨胀，反正钱不是资产，因为钱不可以生钱。

                                    # 2. 负债和资产

                                    负债不一定是坏的，收入看的是现金流，看的是资产产生的金钱减去花费的。

                                    但是负债不在这个公式里面，如果银行愿意你用房子抵押贷款一些钱，然后你用这个房子用来出租

                                    期间你是不要每个月还款的，你的现金流就是收入的。书中提及的游戏讲的就是这样一件事情。

                                    # 3. 财富和资产的定义

                                    • 财富

                                    如果今天停止工作，那么我能活多久。

                                    • 资产

                                    以最低限度的劳动获得钱。

                                    # 4. 资产类型

                                    • 不需要我到场就能够运行的业务

                                    • 股票

                                    • 能产生收入的房地产

                                    • 借据

                                    • 版税

                                    • 有销路的东西

                                    # 5. 财商

                                    财商由四部分组成

                                      1. 会计
                                      1. 投资
                                      1. 市场
                                      1. 法律

                                    这里有一句很有意思的话。

                                    工人付出最大的努力防止工人辞职，雇主提供最少的工资以防止工人辞职。

                                    # 6. 不要让自己有负罪感

                                    负罪感比欲望要糟的多，因为负罪感会抢走灵魂。

                                    # 7. 学会奉献

                                    你想要得到什么，你就越需要给予什么，不是价值的平衡，而是不付出你就不知道你想要得到的东西是怎么样的。

                                    # 8. 克服不良习惯于思维

                                    学习不单单是获取多少知识，更是改掉自己多少的不良习惯和不良思维。

                                    # 9. 钱永远都先给自己

                                    不要害怕负债，不要害怕欠钱。自己花自己该花的，看到自己的账单就逼自己动脑子。

                                    我只能说，变态啊。

                                    # 10. 学会自私，贪婪，接受与拒绝

                                    在这个世界，你不会接受资源，就什么也做不好，因为什么东西都需要用到资源，

                                    学会拒绝永远都是你的人生课题。

                                    # 11. 承认无知，保持清晰。

                                    同样是你，做同一件事情，想有不同的结果是神志不清的表现。做不就应该改变思维，寻找别的方法。

                                    # 12. 在报价的时候，其实你不知道价格，

                                    很多时候，我们的报价都是没有市场调研的，所以其实很多时候，我们和对方很大的可能性都不清楚真正的价格。

                                    顶多就是市场上普遍的价格，但是价格和价值完全不能够混为一谈。
